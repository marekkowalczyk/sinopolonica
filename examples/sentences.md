## Simple Sentences

### Weather (all content words as characters)

| | |
|---|---|
| Polish | Pada deszcz i wieje silny wiatr. |
| Sinopolonica | 降a 雨 與 吹je 強y 風. |
| Gloss | fall-3SG rain and blow-3SG strong-MASC.NOM wind |

### Possession and location

| | |
|---|---|
| Polish | Mój stary ojciec mieszka w małym domu nad rzeką. |
| Sinopolonica | 吾ój 古y 父 居a 中 小ym 家u 臨 河ą. |

### Negation and modality

| | |
|---|---|
| Polish | Nie chcę już pracować w tej fabryce. |
| Sinopolonica | 不 欲ę 已 勞ować 中 此ej 工y. |

### Reflexive and emotional

| | |
|---|---|
| Polish | Dziecko boi się ciemności. |
| Sinopolonica | 童o 懼i 己 闇性i. |

Note: 闇性i = dark + quality + GEN. The suffix *-ość* receives 性
per P03 (see `07-derivation.md`).

### Perfective prefix

| | |
|---|---|
| Polish | Stara kobieta przeczytała długą powieść. |
| Sinopolonica | 古a 女 過読ała 長ą 傳. |

## Complex Sentences

### Total negation

| | |
|---|---|
| Polish | Nikt nigdy niczego nie zrobił. |
| Sinopolonica | 不誰 不時 不何ego 不 z做ił. |

Four instances of 不 make the negative structure transparent.

### Conditional with compound noun

| | |
|---|---|
| Polish | Chciałbym kupić samochód, ale nikt nie chce mi pomóc. |
| Sinopolonica | 欲iał假m 購ić 己行, 但 不誰 不 欲e 吾ie po助c. |

### Complex reflexive

| | |
|---|---|
| Polish | Nikt nie chciał się z nim spotkać, bo bał się go bardzo. |
| Sinopolonica | 不誰 不 欲iał 己 共 彼m 會ać, 因 懼ł 己 彼o 甚. |

### Derivational suffixes in action

| | |
|---|---|
| Polish | Piekarz z piekarni na rogu sprzedaje najlepszy chleb w mieście. |
| Sinopolonica | 焼者 自 焼舍i 上 角u 售aje naj善szy 麵 中 城ie. |

Note: 焼者 (baker = bake + person), 焼舍i (bakery-GEN = bake + place + GEN).
The shared root 焼 (bake) in both words makes the semantic connection
visible.

### Place and abstract quality

| | |
|---|---|
| Polish | Samotność jest najtrudniejsza w ciemności zimowej nocy. |
| Sinopolonica | 獨性 jest naj難iejsza 中 闇性i 冬ej 夜y. |

Note: 獨性 (loneliness = alone + quality), 闇性i (darkness-GEN =
dark + quality + GEN).

## Dialogue

| | |
|---|---|
| Polish | — Gdzie kupiłeś tę książkę? — W księgarni na rogu. |
| Sinopolonica | — 處 購iłeś 此ę 冊ę? — 中 冊舍i 上 角u. |

Note the corrected 冊舍i: book + place + GEN. Both 冊 (book) morphemes
are visible, and the place-suffix 舍 is properly rendered as a character.

## The Full Literary Passage

Polish:

> Szedłem sam przez ciemny las. Wiatr szumiał w starych drzewach.
> Gdzieś daleko szczekał pies.

Sinopolonica:

> 歩łem 獨 過 闇y 林. 風 鳴iał 中 古ych 樹ach.
> 處ś 遠o 吠ał 犬.

## Extended Passage

Polish:

> Wczoraj wieczorem nie mogłem zasnąć, więc wziąłem starą książkę i
> czytałem do trzeciej w nocy.

Sinopolonica:

> 昨 暮orem 不 能łem za眠ąć, 故 wz取ąłem 古ą 冊ę 與
> 読ałem 至 3ciej 中 夜y.
