## Passage 1 — Night Walk (original composition)

### Polish

> Szedłem sam przez ciemny las. Wiatr szumiał w starych drzewach.
> Gdzieś daleko szczekał pies. Myślałem o rzeczach, które dawno
> zapomniałem — o domu dzieciństwa, o głosie ojca, o zapachu chleba
> w kuchni babci. Noc była cicha i pełna gwiazd.

### Sinopolonica

> 歩łem 獨 過 闇y 林. 風 鳴iał 中 古ych 樹ach.
> 處ś 遠o 吠ał 犬. 思ałem o 物ach, które 久o
> 後忘iałem — o 家u 童域a, o 聲ie 父a, o 香u 麵a
> 中 炊舍i 祖母i. 夜 była 靜a 與 滿a 星.

### Gloss notes

| 漢字-Polish | Polish | Analysis |
|---|---|---|
| 思ałem | myślałem | think-1SG.PAST.MASC |
| 物ach | rzeczach | thing-PL.LOC |
| 久o | dawno | long.ago |
| 後忘iałem | zapomniałem | behind-forget-1SG.PAST.MASC |
| 童域a | dzieciństwa | child-domain-GEN |
| 聲ie | głosie | voice-LOC |
| 香u | zapachu | scent-GEN |
| 炊舍i | kuchni | cook-place-GEN |
| 祖母i | babci | grandmother-GEN |
| 靜a | cicha | quiet-FEM.NOM |
| 滿a | pełna | full-FEM.NOM |

### Observations

The passage demonstrates the system's visual rhythm: dense semantic
characters surrounded by clusters of small Latin letters, like stones
in a stream. Polish prepositions (中, 過, o) and conjunctions (與) create
breathing space between the semantic anchors.

The word 炊舍i (kuchni / kitchen-GEN) decomposes *kuchnia* into
cook + place per P03 — the kitchen is "a place where cooking happens,"
which is exactly what 炊舍 says.

The word 童域a (dzieciństwa / childhood-GEN) decomposes *dzieciństwo*
into child + domain per P03 — childhood is "the domain/state of being a
child."

## Passage 2 — Pan Tadeusz, Opening (Adam Mickiewicz)

### Polish (original)

> Litwo! Ojczyzno moja! ty jesteś jak zdrowie;
> Ile cię trzeba cenić, ten tylko się dowie,
> Kto cię stracił.

### Sinopolonica

> Litwo! 父國o 吾a! 汝 存eś 如 康ie;
> 幾 汝ę 須 値ić, 此 唯 己 知ie,
> 誰 汝ę 失ił.

### Gloss notes

| 漢字-Polish | Polish | Analysis |
|---|---|---|
| 父國o | ojczyzno | father-country-VOC |
| 吾a | moja | I-POSS.FEM |
| 存eś | jesteś | be-2SG.PRES |
| 如 | jak | like, as |
| 康ie | zdrowie | health-LOC |
| 幾 | ile | how much |
| 須 | trzeba | must, it is necessary |
| 唯 | tylko | only |
| 値ić | cenić | value-INF |
| 知ie | dowie | know-3SG (perf.) |
| 失ił | stracił | lose-3SG.PAST.MASC |

### Observations

*Ojczyzna* (fatherland) decomposes beautifully into 父 (father) + 國
(country) — the compound logic is identical to many languages' words
for "fatherland" or "motherland." The vocative ending *-o* attaches to
the final character: 父國o.

存eś — the verb *być* (to be) receives 存 (to exist). The suppletive
stems *jest-, by-, będ-* are all absorbed per P09. The conditional
marker 假 (formerly 存 in archaic usage, see `docs/04-verbs.md`) shares
the same etymological root — a fact now visible in the script.

Literary Polish retains its sonic beauty in the Latin-script layer —
the rhyme scheme (zdrowie / dowie) is fully preserved in the suffixes.
The characters add a layer of semantic depth without disrupting prosody.
