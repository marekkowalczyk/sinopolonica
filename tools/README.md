## Planned Tools

This directory is a placeholder for computational tools that would
support the Sinopolonica system.

### Polish-to-Sinopolonica Converter

A tool that takes standard Polish text and converts it to Sinopolonica.
Requirements: morphological analyzer (e.g., Morfeusz for Polish),
character lookup table, rule engine implementing P01–P18.

### Sinopolonica Tokenizer

A tool that splits Sinopolonica text into character-plus-suffix tokens
for indexing, search, and NLP processing.

### Input Method Editor (IME)

A keyboard input method for typing Sinopolonica. Possible approaches:
radical-based input (as in Chinese Wubi/Cangjie), pronunciation-based
input (type Polish, select characters), or hybrid.

### Graded Reader Generator

A tool that takes a Sinopolonica text and a target vocabulary level
(e.g., Core 100, Core 200, Core 500) and produces a version with
glosses for characters above the target level.
