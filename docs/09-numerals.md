## Three-Tier System

Polish currently uses two numeral systems (Arabic and Roman). Sinopolonica
adds a third tier — Chinese numeral characters — for ceremonial and
calligraphic contexts.

| Tier | System | Example | Context |
|---|---|---|---|
| Everyday | Arabic | 1, 2, 3, 15, 2026 | prices, dates, technical, math |
| Formal | Roman | I, II, III, XV, XXI | chapters, centuries, monarchs |
| Ceremonial | Chinese | 一, 二, 三, 十五 | calligraphy, invitations, ritual |

## Usage Contexts

**Arabic (default):** All everyday writing, prices, addresses, phone
numbers, dates, scientific notation, statistics. This is the unmarked
default — use Arabic unless a specific reason calls for another tier.

**Roman (as in current Polish):** Century references (wiek XXI), monarch
ordinals (Jan III Sobieski), chapter numbering in formal publications,
outline levels.

**Chinese (new tier):** Wedding invitations, formal announcements,
commemorative inscriptions, calligraphic works, New Year greetings,
ceremonial documents. Parallel to how Roman numerals add gravitas to
cornerstones and monuments, Chinese numerals add a calligraphic
dimension.

## Chinese Numeral Characters

| Character | Value |
|---|---|
| 一 | 1 |
| 二 | 2 |
| 三 | 3 |
| 四 | 4 |
| 五 | 5 |
| 六 | 6 |
| 七 | 7 |
| 八 | 8 |
| 九 | 9 |
| 十 | 10 |
| 百 | 100 |
| 千 | 1,000 |
| 萬 | 10,000 |
| 〇 | 0 (in digit strings) |

## Example

A wedding invitation might read:

三十 六月 二〇二六年

While the same date on a receipt reads: 30.06.2026.

And a chapter heading: Rozdział III vs. Rozdział 三 (the latter in
calligraphic or ceremonial publication style).
