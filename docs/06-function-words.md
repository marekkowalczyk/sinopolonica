## Governing Principle

Function words that carry semantic content receive characters (A1). The
test is always: "can you explain what this morpheme *means*?" If yes, it
gets a character.

## Prepositions

Prepositions express spatial, temporal, or abstract relationships — they
are semantic. Each receives a character per P01. These same characters
serve as verbal prefixes per P02.

| Character | Polish | Core meaning |
|---|---|---|
| 中 | w | in, inside |
| 上 | na | on, upon |
| 至 | do | to, toward |
| 離 | od | from, away |
| 下 | pod | under, beneath |
| 後 | za | behind |
| 過 | przez | through, across |
| 臨 | nad | above, over |
| 共 | z + instrumental | with, together |
| 自 | z + genitive | from (off of) |

Note: Polish *z* has two distinct meanings depending on the case it
governs. The system disambiguates with two characters — 共 (with, INST)
vs. 自 (from, GEN). This eliminates a genuine source of learner
confusion.

## Modal and Impersonal Predicates

Polish has impersonal modal predicates that carry clear semantic content.
They receive characters per A1:

| Character | Polish | Meaning |
|---|---|---|
| 須 | trzeba | it is necessary, must |
| 可 | można | it is possible, may |
| 宜 | warto | it is worthwhile |
| 宜 | należy | one should (alt. reading) |

## Negation (P17)

Negation is a fundamental semantic operation. The negator *nie* is
**always** written 不. Writing it in Latin script violates A1 and P17.

| 漢字-Polish | Polish |
|---|---|
| 不 欲ę | nie chcę |
| 不 善y | niedobry |
| 不誰 | nikt |
