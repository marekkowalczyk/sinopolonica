## Basic Principle

The character represents the verbal root. Person, tense, gender, and
mood endings attach as Latin suffixes (P04).

## Conjugation

Example: 読 (czytać / to read)

| 漢字-Polish | Polish | Analysis |
|---|---|---|
| 読ać | czytać | read-INF |
| 読am | czytam | read-1SG.PRES |
| 読asz | czytasz | read-2SG.PRES |
| 読a | czyta | read-3SG.PRES |
| 読ał | czytał | read-3SG.PAST.MASC |
| 読ała | czytała | read-3SG.PAST.FEM |
| 読ałem | czytałem | read-1SG.PAST.MASC |
| 読ałam | czytałam | read-1SG.PAST.FEM |

## Multi-Reading (P09)

Suppletive verb stems are absorbed by the character. The reader infers
pronunciation from the ending.

Example: 歩 (iść / chodzić — to go / to walk)

| 漢字-Polish | Polish | Absorbed stem |
|---|---|---|
| 歩ę | idę | id- |
| 歩łem | szedłem | szed- |
| 歩łam | szłam | sz- |
| 歩odzić | chodzić | chodz- |
| 歩odzę | chodzę | chodz- |

Example: 存 (być — to be / to exist). This is Polish's most
fundamental verb and one of its most suppletive:

| 漢字-Polish | Polish | Absorbed stem |
|---|---|---|
| 存em | jestem | jest- |
| 存eś | jesteś | jest- |
| 存 | jest | jest (zero ending) |
| 存ł | był | by- |
| 存ła | była | by- |
| 存ędzie | będzie | będ- |
| 存ędziemy | będziemy | będ- |

Four suppletive stems (jest-, jes-, by-, będ-) absorbed by a single
character. Note the connection to the conditional marker: the archaic
aorist of 存 fossilized into the conditional clitic, now written 假 in
modern standard (see "The Conditional Marker" below).

Example: 取 (jąć — to seize, archaic root per P11)

| 漢字-Polish | Polish | Compound |
|---|---|---|
| 取ąć | jąć | seize (archaic) |
| wz取ąć | wziąć | PFV-seize = take |
| u取ąć | ująć | grasp, charm |
| 下取ąć | podjąć | under-seize = undertake |
| ob取ąć | objąć | embrace |
| 自取ąć | zdjąć | from-seize = remove |

## Verbal Prefixes (P02)

Polish verbal prefixes carry spatial/aspectual meaning. Each receives the
same character as its corresponding preposition:

| Character | As preposition | As prefix | Spatial meaning |
|---|---|---|---|
| 過 | przez (through) | prze- | through, completion |
| 出 | — | wy- | out, extraction |
| 至 | do (to) | do- | toward, completion |
| 離 | od (from) | od- | away, reversal |
| 入 | — | w- | in, entry |
| 上 | na (on) | na- | onto, perfective |
| 下 | pod (under) | pod- | beneath, subsidiary |
| 後 | za (behind) | za- | behind, recording |

### The 書 (write / -pis-) family

| 漢字-Polish | Polish | Literal |
|---|---|---|
| 書ać | pisać | write (impf.) |
| 上書ać | napisać | on-write (perf.) |
| 下書ać | podpisać | under-write = sign |
| 後書ać | zapisać | behind-write = note down |
| 過書ać | przepisać | through-write = copy |
| 出書ać | wypisać | out-write = list out |
| 至書ać | dopisać | to-write = append |
| 離書ać | odpisać | away-write = reply |
| 入書ać | wpisać | in-write = inscribe |

### Prefix stacking

Polish permits multiple prefixes. Each semantic prefix gets a character;
purely aspectual prefixes stay in Latin:

| 漢字-Polish | Polish | Analysis |
|---|---|---|
| po-離轉ała | poodkręcała | DISTR-away-turn-PAST.FEM |

Here *po-* is purely distributive (no spatial content) → Latin. 離 (od-
/ away) and 轉 (turn) are semantic → characters.

## The Conditional Marker (P12)

The Polish conditional clitic *-by-* is a fossilized aorist of *być* (to
be). The system preserves the historical layering:

| Period | Form | Character | Logic |
|---|---|---|---|
| Archaic | 欲iał 存 jesm | 存 | aorist of "to be" |
| Transitional | 欲iał存m | 存 | fusing |
| Modern standard | 欲iał假m | 假 | hypothetical/conditional |

Modern standard uses 假 (hypothetical). The archaic 存 (to exist) may
appear in deliberately archaizing texts.

Example:

| 漢字-Polish | Polish |
|---|---|
| 欲iał假m 購ić 己行 | chciałbym kupić samochód |

## The Jussive (令)

The jussive particle *niech* receives 令 (let, cause):

| 漢字-Polish | Polish |
|---|---|
| 令 活je 王! | Niech żyje król! |
