## Basic Principle

The character represents the noun's semantic root (P01). Polish case
endings attach directly as Latin-script suffixes (P04). A bare character
with no suffix represents the nominative when the nominative has a zero
ending.

## Case Declension

Example: 家 (dom / house)

| 漢字-Polish | Polish | Case |
|---|---|---|
| 家 | dom | NOM (zero ending) |
| 家u | domu | GEN / LOC |
| 家owi | domowi | DAT |
| 家 | dom | ACC (zero, masc. inanimate) |
| 家em | domem | INST |
| 家u | domu | LOC |

Example: 河 (rzeka / river)

| 漢字-Polish | Polish | Case |
|---|---|---|
| 河a | rzeka | NOM |
| 河i | rzeki | GEN |
| 河ę | rzekę | ACC |
| 河ą | rzeką | INST |

Note: the stem alternation *rzek- / rzec-* is absorbed by 河 per P09.

## Zero-Ending Nominatives

When the nominative singular has a zero ending, the character stands
alone — exactly as in Classical Chinese. This creates moments of pure
logographic writing within the hybrid system.

| 漢字-Polish | Polish |
|---|---|
| 林 | las |
| 犬 | pies |
| 風 | wiatr |
| 雨 | deszcz |

## Diminutives and Augmentatives

Diminutive and augmentative suffixes are derivational but their semantic
contribution is scalar (size/affect) rather than categorical. They remain
in Latin script per P04. The character stays stable:

| 漢字-Polish | Polish | Analysis |
|---|---|---|
| 家 | dom | house |
| 家ek | domek | house-DIM |
| 家eczek | domeczek | house-DIM-DIM |
| 家isko | domisko | house-AUG |
| 犬ek | piesek | dog-DIM |
| 犬eczek | pieseczek | dog-DIM-DIM |
